Principio: raccolta minima dei dati

Questo sito raccoglie il minor numero possibile di dati personali. Non utilizziamo cookie di tracciamento, strumenti di analisi (come Google Analytics) né pubblicità.

Hosting e file di log del server

Quando visiti questo sito, il fornitore di hosting raccoglie automaticamente dati tecnici (i cosiddetti file di log del server). Tra questi:

  • Indirizzo IP (anonimizzato)
  • Data e ora dell'accesso
  • Pagina richiamata
  • Browser e sistema operativo

Questi dati non vengono uniti ad altre fonti di dati e vengono cancellati automaticamente dopo 7 giorni. La base giuridica è l'Art. 6 Abs. 1 lit. f DSGVO (interesse legittimo alla fornitura tecnica del sito).

Caratteri (integrati localmente)

Tutti i caratteri di questo sito vengono forniti localmente dal nostro server. Non esiste alcun collegamento con Google Fonts o altri fornitori esterni. Durante il caricamento dei caratteri non vengono trasmessi dati a terzi e il tuo indirizzo IP non viene comunicato a server di font esterni.

Progressive Web App (PWA)

Questo sito può essere installato come app sul tuo dispositivo. In tal caso i contenuti vengono salvati localmente sul tuo dispositivo, così puoi usare la pagina anche senza connessione a internet. In questo processo nessun dato personale viene trasmesso a noi.

Funzione di lettura ad alta voce

La funzione di lettura ad alta voce utilizza la sintesi vocale del tuo browser (Web Speech API). I testi vengono elaborati localmente sul tuo dispositivo e non vengono inviati a server esterni.
