# Tools und Werkzeuge für Fachkräfte

Werkzeuge für Digitale Ersthelfer, CSN-Praktikerinnen und Vorfallexperten.
Alle hier aufgeführten Tools sind frei verfügbar.

> Für den Einsatz in rechtlichen Verfahren sollte immer eine qualifizierte
> IT-Forensikerin oder ein qualifizierter IT-Forensiker hinzugezogen werden.

---

## Ransomware und Verschlüsselung

| Tool | Beschreibung | URL |
|---|---|---|
| **No More Ransom** | Kostenlose Entschlüsselungstools (Europol-Projekt) | [nomoreransom.org](https://www.nomoreransom.org/de/) |
| **ID Ransomware** | Ransomware-Typ anhand verschlüsselter Datei identifizieren | [id-ransomware.malwarehunterteam.com](https://id-ransomware.malwarehunterteam.com) |

## Phishing und E-Mail-Analyse

| Tool | Beschreibung | URL |
|---|---|---|
| **VirusTotal** | Dateien und URLs auf Malware prüfen | [virustotal.com](https://www.virustotal.com) |
| **MXToolbox** | E-Mail-Header analysieren | [mxtoolbox.com](https://mxtoolbox.com/EmailHeaders.aspx) |
| **PhishTank** | Bekannte Phishing-URLs nachschlagen | [phishtank.org](https://phishtank.org) |
