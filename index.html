Unbeobachtet surfen

Sind Sie sicher? Tipps zum unbeobachteten Surfen
  • Nutzen Sie möglichst ein Gerät, auf das die Person keinen Zugriff hat, vor der Sie sich schützen möchten — zum Beispiel das Handy einer Vertrauensperson oder einen Computer in der Bücherei.
  • Privates Fenster (Inkognito) öffnen: Drücken Sie Strg + Umschalt + N und öffnen Sie diese Seite dort erneut. So bleiben weniger Spuren auf dem Gerät zurück.
  • Verlauf löschen: Mit Strg + Umschalt + Entf öffnen Sie das Lösch-Fenster Ihres Browsers. Bedenken Sie: Wenn plötzlich der komplette Verlauf fehlt, kann auch das auffallen — löschen Sie im Zweifel nur den Zeitraum „letzte Stunde".
  • Diese Webseite selbst speichert nichts über Sie — keine Cookies, kein Tracking. In Ihrem Browser-Verlauf bleibt aber der Seitenaufruf sichtbar, bis Sie ihn löschen.
  • Mit dem Button „Schnell verlassen" (unten links) oder zweimal Esc wechseln Sie sofort zu einer neutralen Seite.
